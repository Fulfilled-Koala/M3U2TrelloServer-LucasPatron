# Trello Clone Express Server

Express server with TypeScript for the Trello Clone project. PORT is set to 1337.

## Setup

Cloning the project:

```bash
$ git clone
```

Running the server:

```bash
$ npm start
```

Building the project:

```bash
$ npm run postinstall
```
